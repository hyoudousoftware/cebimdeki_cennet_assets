# Cebimdeki Cennet Uygulama İçi Varlık Dosyaları

Bu repo, Cebimdeki Cennet mobil uygulaması için kullanılan asset varlık dosyalarını merkezi olarak saklamak amacıyla oluşturulmuştur.

## 📝 İçerik

- `images/` klasörü altında uygulama görselleri bulunur.

## ℹ️ Kullanım

Bu depodaki dosyalar, mobil uygulama geliştirme sürecinde aşağıdaki amaçlarla kullanılabilir:

- Uygulama içi görsel kaynakların yönetimi
- Ekranlarda tekrar kullanılabilir assetlerin paylaşımı
- Proje ekipleri arasında ortak asset havuzu oluşturulması
