# cebimdeki_cennet_assets

Bu repo, cebimdeki_cennet mobil uygulaması için kullanılan asset varlık dosyalarını merkezi olarak saklamak amacıyla oluşturulmuştur.

## İçerik

- `images/` klasörü altında uygulama görselleri ve varlık dosyaları bulunur.
- `images/login_logos/` klasörü, giriş ekranı logo varlıkları için ayrılmıştır.

## Kullanım

Bu depodaki dosyalar, mobil uygulama geliştirme sürecinde aşağıdaki amaçlarla kullanılabilir:

- Uygulama içi görsel kaynakların yönetimi
- Giriş ekranı ve diğer ekranlarda tekrar kullanılabilir assetlerin paylaşımı
- Proje ekipleri arasında ortak asset havuzu oluşturulması
